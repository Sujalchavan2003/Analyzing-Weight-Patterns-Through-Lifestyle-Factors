# Analyzing-Weight-Patterns-Through-Lifestyle-Factors
A Python-based EDA project exploring the impact of lifestyle habits like diet, alcohol consumption, transportation, and family history on weight and obesity levels.


## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Gender distribution vs weight
- Family history of overweight vs weight
- High-calorie food consumption vs weight
- Alcohol consumption vs weight
- Transportation mode vs weight
- Obesity level distribution

## Key Insights
- Family history is associated with higher body weight
- Frequent high-calorie food consumption increases obesity risk
- Active transportation is linked to lower average weight
- Obesity is influenced by multiple interacting lifestyle factors
